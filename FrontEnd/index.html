SpamShield IA

Início Projeto Tecnologias Contato

Detector Inteligente de Spam com Machine Learning

Projeto acadêmico desenvolvido em Python utilizando Inteligência Artificial para identificar automaticamente mensagens de spam em emails.

Exemplo de análise

“Parabéns! Você ganhou um prêmio.”
Resultado: SPAM
“Oi, vamos marcar a reunião amanhã?”
Resultado: HAM

Sobre o Projeto

O sistema utiliza técnicas de Machine Learning e Processamento de Linguagem Natural para classificar mensagens como spam ou não spam.

Pré-processamento

Limpeza e preparação dos dados utilizando Pandas.

TF-IDF

Conversão de textos em valores numéricos para análise da IA.

Naive Bayes

Algoritmo de Machine Learning utilizado para classificação.

Tecnologias Utilizadas

Python HTML5 CSS3 Pandas Scikit-learn Machine Learning

Projeto acadêmico de detecção de spam utilizando Machine Learning e simulação com JavaScript.
